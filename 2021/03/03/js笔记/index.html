第一章 JavaScript的基础语法

一，js定义（默写）

js是基于对象和事件的编程语言。

优点：js是解释性语言 脚本语言 跨平台性好

缺点：响应慢

二，计算机语言

计算机语言分为 机器语言，汇编语言，高级语言（编程语言）

编程语言—>翻译器—–>机器语言

翻译分为编译和解释两种，编译会生成一个文件，速度块，跨平台性不好，如C ,C++，解释性语言会边解释边执行，跨平台性好，速度稍慢，如：js,PHP

三，JavaScript组成（默写）

js分为ECMAScript（语法标准）,BOM：Browser Object Model（浏览器对象模型）,DOM： Document Object Module （文档对象模型）。

四，变量（默写）

1.定义

用于存储信息的容器。

2.声明赋值

用var关键字声明，可以先声明后赋值，也可以同时声明和复制。

3.取名规范

由数字，字母，$，_组成，不能以数字开头，不能使用关键字过着保留字，注意大小写。

4.作用域

1.全局变量 声明在函数外面，全局有效，始终保存在内存中。

2.局部变量 声明在函数内，只有在函数内有效，只有在函数调用时候在存储在内存中，函数调用结束时存储的内容也会被销毁。

第三章 函数

函数的定义：程序的基本单元，是完成特定任务的代码语句块。是为实现一定功能的代码语句块。

函数的特征

  • 实现一定的功能
  • 可以有参数
  • 可以返回一个结果

函数分类：系统函数（内置函数），自定义函数（声明函数，封装函数）

系统函数

自定义函数

一，以是否又函数名，分为有名函数/具名函数，匿名函数。

①单独声明一个函数，必须写函数名，函数名取名规范同变量的取名规范。

②function左边有=，这个函数就是匿名函数。

③函数声明在哪里不重要，函数在哪里调用就在那里生效。

④函数调用后才生效，函数名（）

二，以函数有无参数，分为有参函数（带参函数），无参函数。

①有不确定的内容需要传参。

②参数分为形参，实参

  • 形参：声明函数时的参数，本质没有变量，没传实参的形参返回值时undefined
  • 实参：调用参数时的函数，实参是确定的值
  • 方向：把实参传递给形参
  • 顺序&个数：实参的顺序和形参保持一致 ，个数也尽量保持一致

arguments（实参列表）：类似数组的东西。

三，以函数有无返回值，分为有返回值函数，无返回值函数。

①函数声明内有return 这个函数调用后的返回值使return后的内容，没有return的为undefined

一.通过DOM树/层级关系找到节点

Ⅰ.查找节点

1.子节点.parentNode //找父节点 父节点.childNodes //返回子节点集合 childNodes[i]

2..父节点.firstElementchild //找第一个标签子节点

5.弟节点.previousElementSibling //找紧跟弟节点的兄节点

1.document.createElement(“标签名”) //通过标签名来创建标签

2.被克隆节点.cloneNode(deep) //deep（传true）深度复制**克隆节点本身以及里面所有后代** 不传参表示克隆节点本身

注意：创建的，克隆的标签都要添加到页面后才能看到

3.父节点.appendChild(子节点) //将子节点添加到父节点末尾

3.父节点.insertBefore(兄节点(new)，弟节点(old)) //将**兄节点添加到弟节点**前面

Ⅲ.删除

1.父节点.removeChild(子节点) //从父节点中删除子节点 父节点始终存在

2.节点.remove() //删除当前节点及里面内容
