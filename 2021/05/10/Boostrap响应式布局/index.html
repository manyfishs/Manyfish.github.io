Boostrap响应式布局

Bootstrap 是一个基于 HTML、CSS、JAVASCRIPT 用于快速开发 Web 应用程序和网站的开源的前端框架。
移动设备优先：自 Bootstrap 3 起，框架包含了贯穿于整个库的移动设备优先的样式。
目前所有的主流浏览器都支持 Bootstrap
它 的响应式 CSS 能够自适应于台式机、平板电脑和手机 贯彻了移动现行的宗旨
它 提供了一个带有网格系统、链接样式、背景的基本结构
它由Twitter的设计师Mark Otto和Jacob Thornton合作开发

Boostrap_4和Boostrap_3的区别

Bootstrap 3 和 Bootstrap 4 区别在于 Bootstrap 4 现在使用 flexbox（弹性盒子） 而不是浮动

bootstrap4源码是采用 Sass 语言编写；
bootstrap3源码是采用 Less 语言编写

Bootstrap 4 共有5种栅格类，特小(col-)、小(col-sm-)、中(col-md-)、大(col-lg-)、特大(col-xl-)
bootstrap3 共有4种栅格类，特小(col-xs-)、小(col-sm-)、中(col-md-)、大(col-lg-)

Bootstrap4拥有了更多的具体的把一些有关的部分变成了相关的组件。
Bootstrap4的Bootstrap.min.css的体积减少了40%以上。
Bootstrap4放弃了对iE8以及ios 6的支持，现在仅仅支持IE9以上以及iOS7以上版本的浏览器。如果要支持以前的浏览器，要用Bootstrap3
