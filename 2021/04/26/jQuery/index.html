jQuery基础

jQuery 是一个 JavaScript 库。对js对象和使用函数的封装

jQuery 极大地简化了 JavaScript 编程。

jQuery 很容易学习。

jQuery由美国人John Resig于2006年创建 设计思想：write less do more

jQuery能做什么

1.方便快捷获取DOM元素

2.动态修改页面样式

3.动态改变DOM内容

4.响应用户的交互操作

5.为页面添加动态效果

6.统一Ajax操作

7.简化常见的JavaScript任务。

元素的尺寸

对元素的宽高进行设置

jQuery方法 描述
节点对象.width()/节点对象.height() 取得匹配元素宽度和高度值，只包含 width/height
节点对象.innerHeight()/节点对象.innerWidth() 取得匹配元素宽度和高度值，包含 width/height 、padding
节点对象.outerHeight()/节点对象.outerWidth() 取得匹配元素宽度和高度值，包含 width/height 、padding、border
节点对象.outerHeight(true)/节点对象.outerWidth(true) 取得匹配元素宽度和高度值，包含 width/height 、padding、border、margin

以上参数为空，表示获取相应值，返回的时数字类型

有参数且为数字类型，表示修改相应值。

元素的位置

jQuery方法 描述
节点对象.offset().left 获取距离文档左侧的距离
节点对象.offset().top 获取距离文档顶部的距离
节点对象.position().left 获取距离定位父级左侧距离
节点对象.position().top 获取距离定位父级顶部侧距离
节点对象.scrollTop() 获取元素被卷去的头部
节点对象.scrollLeft() 获取元素被卷去的左侧

offset ()：方法设置或者返回元素相对于文档的偏移坐标，通过属性 left、top来获取，通过 **节点对象.offset( {top:10,left:30} )**的形式设置偏移量。

position():方法用于返回元素相对于带有定位的父级偏移坐标，父级没有定位，就以文档为准，该方法只能获取。

scrollTop()、scrollLeft():不带参数表示获取，有参数表示设置被卷去的位置。

加载函数

相同点：都是放在head内且要获取节点，需要加加载函数。
不同点： 1. js的文件只需要放在head内即可，jq的文件要放在库文件的下面。
2. js没有简写形式，jq有简写形式。

4. js加载函数需要等文档完全加载完成才能执行，jq只需要必要的内容加载完毕就可以执行。

jQuery事件注册

jQuery为我们提供了方便的事件注册机制，优缺点如下:

  • 优点:操作简单，且不用担心事件覆盖等问题。

  • 缺点:普通的事件注册不能做事件委托，且无法实现事件解绑，需要借助其他方法。

语法:

其他事件和原生基本一致。
比如mouseover、mouseout、blur、focus、change.keydown、keyup、resize、scroll等

一.jQuery事件处理

因为普通注册事件方法的不足，jQuery又开发了多个处理方法

. on():用于事件绑定，目前最好用的事件绑定方法
. off():事件解绑
. trigger() / triggerHandler():事件触发

1.事件处理on()绑定事件

因为普通注册事件方法的不足，jQuery又创建了多个新的事件绑定方法bind() / live() / delegate() / on()等，其中最好用的是: on()

语法
on()方法优势1:
可以绑定多个事件，多个处理事件处理程序。

如果事件处理程序相同

on()方法优势2:
可以事件委派操作。事件委派定义是，把原来加给子元素身上的事件绑定在父元素身上，就是把事件委派给父元素。

在此之前有bind(). live() deleaate()等方法来处理事件绑定或者事件委派，最新版本的请用on替代他们.

on()方法优势3:
动态创建的元素，click(没有办法绑定事件, on)可以给动态生成的元素绑定事件

2.事件处理off()解绑事件

当某个事件上面的逻辑，在特定需求下不需要的时候，可以把该事件上的逻辑移除，这个过程我们称为事件解绑。jQuery为我们提供了多种事件解绑方法: die() / undelegate() / off()等，甚至还有只触发一次的事件绑定方法one()

语法

off()方法可以移除通过on()方法添加的事件处理程序。

如果有的事件只想触发一次，可以使用one()来绑定事件。

二.jQuery事件对象

jQuery对DOM中的事件对象event进行了封装，兼容性更好，获取更方便，使用变化不大。事件被触发，就会有事件对象的产生。

语法

阻止默认行为: event.preventDefault() 或者return false
阻止冒泡: event.stopPropagation()

注意:jQuery中的event对象使用，可以借鉴API和DOM中的event 。

三.事件切换

jQuery中添加了一个新事件 hover();功能类似css 中的伪类:hover。

语法

  • over:鼠标移到元素上要触发的函数 (相当于mouseenter)
  • out:鼠标移出元素要触发的函数 (相当于mouseleave)
  • 如果只写一个函数，则鼠标经过和离开都会触发它
